# TooManyBodies
时不时被攒了一地的盒子卡手卡得窝火？

一键清理舔空了的盒子，妈妈再也不用担心满地烤鸡吃不完啦

默认B键，Mod排序在Modconfig模组后可以调整设置

有BUG或者问题可以留言


Tired of stumbling over piles of empty loot boxes cluttering your path?

One-click cleanup for all those looted boxes—no more tripping over half-eaten chicken legs everywhere!

Default key: B. Adjust settings in ModConfig if the mod loads after it.

Feel free to report any bugs or issues in the comments.
